# services-rest-api

Rest API template with Node.js, Express and and MongoDB with Mongoose. Featuring authentication, user management and rental of services.

Endpunkte:

- /api/users
- /api/services
- /api/categories
- /api/subscriptions
- /api/returns
- /api/auths

API hosted at Heroku: https://services-rest-api.herokuapp.com/
